# 数字图像处理应用程序

这是一个功能强大的数字图像处理应用程序，基于Python开发，提供了丰富的图像处理和编辑功能。通过直观的用户界面，用户可以轻松地对图像进行各种处理和优化。

## 🌟 主要功能

### 基础图像操作
- **图像缩放**：支持10%-200%的精确缩放，保持图像质量
- **打开/保存图像**：支持多种常见图像格式的导入和导出
- **重置图像**：一键恢复原始图像状态

### 颜色调整
- **灰度转换**：将彩色图像转换为灰度图像
- **反色处理**：创建图像的负片效果
- **饱和度增强**：调整图像的色彩饱和度
- **亮度和对比度**：精细调整图像的亮度和对比度

### 图像滤镜
- **模糊效果**：应用高斯模糊等模糊滤镜
- **锐化处理**：增强图像细节和边缘
- **边缘检测**：突出显示图像中的边缘
- **特殊效果**：提供多种艺术滤镜效果

## 📋 系统要求

- **操作系统**：Windows/macOS/Linux
- **Python版本**：Python 3.8+
- **依赖包**：
  - PyQt6 (用户界面)
  - OpenCV (图像处理)
  - Pillow (图像操作)
  - NumPy (数值计算)

## 🚀 安装指南

1. 确保您的系统已安装Python 3.8或更高版本
2. 克隆或下载本仓库到本地
3. 安装所需依赖包：
   ```bash
   pip install -r requirements.txt
   ```

## 📖 使用说明

### 启动应用程序

```bash
cd code
python main.py
```

### 基本操作流程

1. **导入图像**：点击「打开图片」按钮选择要处理的图像文件
2. **调整图像**：
   - 使用滑块调整图像缩放比例
   - 从下拉菜单中选择颜色调整效果
   - 从下拉菜单中选择滤镜效果
3. **保存结果**：点击「保存图片」按钮将处理后的图像保存到本地

## ⚠️ 注意事项

- **支持的图像格式**：PNG, JPG, XPM, BMP
- **性能考虑**：处理超大分辨率图像可能会影响应用性能，建议使用适中大小的图像
- **保存格式**：根据需要选择合适的输出格式，不同格式有不同的压缩特性

pyinstaller --noconfirm --windowed --icon="code/icon/app.jpg" --add-data "code/icon;icon" --add-data "code/Zero_DCE;Zero_DCE" code/main.py
